About

ALOLA! I am Dev (yeah, Dev is my name >_>). I am an indie developer working on aerial ace as a side project. It was the first ever discord bot I made. It started as a small project so that I can learn how discord bots worked.

I have always been a huge pokemon fan, and was spending a lot of my time playing poketwo :| After playing poketwo for quite a while, I decided to start the development of this bot. This bot was (for me) a way to both learn bot development and simultaneously make something for the community that i been into for such long a time.

I am a game developer too... Well, never released anything crazy but I have made some small games and prototypes. You probably don't want to see them, but links are below, just in case :]

BTW, are you an anime enthusiast? I am also working on YUI BOT which is an all-in-one Anilist companion bot. Make sure to give feedback if you check it out.
