All Commands

Here is the list of all the available commands with examples.

Help Command : -aa help

Pokedex

All commands related to pokedex.

Command Description Parameters Example
Dex Provides the pokedex entry of any pokemon Pokemon -aa dex durant

Battle Details

All commands related to battles and dueling

Command Description Parameters Example
Stats Provides the compulsary stats of any Duelish Pokemon Pokemon -aa stats solgaleo
Moveset Provides the best moveset for any Duelish Pokemon Pokemon -aa moveset regigigas
Nature Provides the best nature for any Duelish Pokemon Pokemon -aa nature kyurem
Weakness Provides the type weakness of any pokemon or type(s) Pokemon/Type(s) -aa weakness mew
-aa weakness ice water
Tierlist Provides the tierlist of the pokemon sorted by types Tier -aa tierlist steel
-aa tl mega

Shiny Pings

All commands related to shiny pings.

Command Description Parameters Example
Tag Adds the user to the provided tag ping list Pokemon -aa tag aron
Tag Ping Pings all users assigned to the provided tag Pokemon -aa tag_ping pichu
Tag Show Shows all users assigned to the provided tag without pinging. Pokemon -aa tag_show morpeko
AFK Prevents the user from getting pinged from tag_ping. State -aa afk on
-aa afk off
Admin Commands
Tag Remove Removes the user from their Tag Member -aa tr @P5
Tag Remove ID Removes the user from their Tag using user_id User ID -aa trid 716390085896962058
